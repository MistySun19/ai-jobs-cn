# AI 对中国就业市场的冲击

交互式可视化：分析 AI 对中国 ~150 个职业的暴露程度。

**[在线演示 →](https://mistysun19.github.io/ai-jobs-cn/)**

![preview](preview.png)

## 概述

本项目受 [Andrej Karpathy 的美国就业市场 AI 暴露分析](https://joshkale.github.io/jobs/) 启发，针对中国劳动力市场进行了本土化改编。

使用 Treemap（矩形树图）可视化中国约 150 个职业的 AI 暴露程度：
- **面积** = 就业人数
- **颜色** = AI 暴露评分（绿色→安全，红色→高暴露）

## 数据来源

### 就业人数

| 来源 | 覆盖范围 |
|------|----------|
| 第五次全国经济普查公报（2024 年发布，统计时点 2023 年末） | 20 个行业从业人员总数，作为行业级基数 |
| 教育部 2024 年全国教育事业发展统计公报 | 幼儿教师、小学/中学/高校教师等 7 个教育职业 |
| 国家卫健委 2024 年卫生健康事业发展统计公报 | 临床医生、注册护士 |
| 司法部 / 律师协会 / 中注协 / 中注税协 | 律师、公证员、法官、注册会计师、税务师等 |
| 交通运输部 / 国家邮政局 / 新华网 | 网约车司机、快递员、外卖骑手 |
| 公安部 / 应急管理部 / 人社部 | 公务员、公安民警、消防员 |
| 中国民航局 2024 年民航行业发展统计公报 | 民航飞行员 |

行业内各职业的就业人数，由行业总人数按《中华人民共和国职业分类大典（2022 版）》中类权重拆分得到。

### 薪资

- **广东省 2023 年企业薪酬调查数据**（128 个职业的 P25/P50/P75 分位薪酬），通过校准系数（×0.75）折算为全国估值
- **国家统计局 2024 年城镇单位就业人员年平均工资**（非私营 55% + 私营 45% 加权），用于行业级约束

### AI 暴露评分

- 0–10 分，由 Claude（claude-sonnet-4-6）基于职业描述逐一评分
- 评分维度：工作产出的数字化程度、核心任务是否可被 LLM/代码生成/自动化替代
- 辅以启发式规则校准（按职业大类基础分 + 关键词调整）

## AI 暴露评分标准

| 分数 | 等级 | 示例职业 |
|------|------|----------|
| 0-1 | 极低 | 建筑工人、清洁工、农民 |
| 2-3 | 低 | 电工、水管工、护士、消防员 |
| 4-5 | 中等 | 教师、零售、医生 |
| 6-7 | 较高 | 管理人员、工程师、会计 |
| 8-9 | 很高 | 软件开发、金融分析师、翻译、编辑 |
| 10 | 极高 | 数据录入员 |

核心判断依据：**工作产出是否本质上是数字化的** — 如果一份工作可以完全在电脑前完成，AI 暴露天然就高。

本项目仅供参考和讨论，不构成任何职业建议。AI 暴露评分由 AI 模型生成，存在固有偏差。实际的 AI 对就业的影响远比单一数字复杂。
